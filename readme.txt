Add an editor button to your admin bar.

== Description ==

Since WordPress Version 3.1 - you know - there is the admin bar. Through this plugin you will add a link to the theme editor.

== Installation ==

Easily upload the plugin into you plugin directory (/wp-content/plugins/) and activate the plugin through the 'Plugins' menu in WordPress.
